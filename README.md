# Blog API

A RESTful API for managing blog posts and users. This project simulates a blog-focused social media platform where users can create and manage content securely.

## Features
- User registration and authentication using JWT
- Create, update, and delete blog posts
- Manage user accounts
- Public access to read blog posts and view user information
- Persistent data storage using a relational database

## Tech Stack
- Kotlin
- Spring Boot
- PostgreSQL
- JWT Authentication
- Docker
